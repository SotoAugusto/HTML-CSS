Relative and Absolute URLs

Relative

  • within one domain/ website: relative to one resource, where is the other resoruce
    • example:
    • pic/anatomy-of-an-html/element.png
    • ...pic/anatomy-of-an-html-elemment.png
the src attribute of this image is using a relative url ,"coffee.jpg"

the href attribute in the link tag is a relative link too

the src attribute of this image is using a relative url ,"img/triangles.png"

Absolute

  • full url
    • https://docs.emmet.io/cheat-sheet/
  • from the root of a site, where is the resource
    • If the path part of the URL starts with the "/" character, the browser will fetch that resource from the top root of the server, without reference to the context given by the current document.
    • /pic/anatomy-of-an-html-element.png
