the :link pseudoclass targets the following anchor, making in so that changes the default color of the link, in this case, to blue.

go to google

the next pseudo class is :visited, which targets the anchor once it gets clicked, in this case changes the color to a green.

:hover targets the anchor when you hover the mouse over the text, in this case changes the color to darkorange

:active targets the anchor in the time between you press the mouse click and release it, in this case changes the color to greenyellow
